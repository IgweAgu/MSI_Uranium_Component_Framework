Introduction

Uranium is a unique design system introduced for Orchestrate (ConnectX). A web application used for law enforcement professionals and public safety men and women. The system was designed with UX (User Experience) as the primary focus.


With a fast, rapid and fluid UI (User Interface) methodology incorporated for adaptive multi-functional platforms such as web, mobile and tablet. Uranium is created with a more steam-line approach to front-end development and pre-existing technologies such as HTML5 & Angular.


At its core, it values scalability and user-driven actions to accommodate API predictability and support backend architecture with a constant response to web integrations. View how orchestrate uses uranium to create a responsive & dynamic user-driven and state management UI
